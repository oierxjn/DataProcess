# 这个项目是什么
用来快捷地帮助同济的学子处理物理实验中的数据。

项目中对特定的实验提供了数据处理、图标绘制。

# 如何使用
左上角切换到对应的实验分支，再点右边的code按钮，即可下载对应的zip包。

或者点这里快速切换：
